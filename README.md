📊 Project Overview

This project presents an interactive Super Store Sales Dashboard created using Microsoft Power BI.
The dashboard helps analyze sales performance, profit trends, shipping modes, payment methods, and regional sales distribution.

It provides business insights through visualizations such as line charts, bar charts, maps, and donut charts.

🎯 Objective

The objective of this dashboard is to:

Analyze sales and profit trends over time

Understand category and sub-category performance

Identify regional sales distribution

Compare shipping modes and payment methods

Provide insights for better business decision making

📌 Dashboard Features

The dashboard includes the following key metrics and visuals:

Total Sales: 450.23K

Total Quantity Sold: 6251

Total Profit: 53.40K

Average Delivery Time: 3.91 days
